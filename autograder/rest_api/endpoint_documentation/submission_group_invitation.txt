/submission_group_invitations/<id>/
GET: Returns the specified group request.
Access rules:
    - If the user making the request is not the invitation creator or one of the
        invitees, then 403 Forbidden will be returned.
    - If the user making the request doesn't have permission to view the 
        project the specified group is associated with, 403 Forbidden will be returned.

Response format:
{
    "type": "submission_group_invitation",
    "id": <id>,
    "request_creator": <string username>,
    "invited_members_to_acceptance": {
        "<username>": <bool>,
        ...
    },
    "semester_name": <string>,
    "project_name": <string>,
    "urls": {
        "self": "/submission_group_invitations/<id>/",
        "accept": "/submission_group_invitations/<id>/accept/",
        "project": "/projects/<id>/"
    }
}

DELETE: Rejects the submission group invitation by deleting it and sending
notifications to the other users involved that the invitation was rejected.
Access rules:
    - The user making the request must either be the invitation creator or
        one of the invitees, otherwise 403 Forbidden will be returned.
    - If the user making the request doesn't have permission to view the 
        project the specified group is associated with, 403 Forbidden will be returned.

/submission_group_invitations/<id>/accept/
POST: Marks the specified submission group invitation as accepted for the 
user making the request. If all invited members have accepted the invitation,
then the invitation will be deleted and replaced with a submission group.
Notifications will also be sent to the users involved that the invitation was
accepted.

Access rules:
    - If the user making this request is not one of the invited members, 
        403 Forbidden will be returned.
    - If the user making the request doesn't have permission to view the 
        project the specified group is associated with, 403 Forbidden will be returned.

The request and response have no content for this endpoint. 
On success, 204 No content will be returned.
